ModelSelectionGUI.OPEN_BROWSER_DEFAULT — Constant
OPEN_BROWSER_DEFAULT

Default value for whether to open the browser automatically when the server starts for Swagger documentation. The default behavior is not to open the browser, i.e., false.

source
